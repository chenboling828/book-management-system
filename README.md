# 简易图书管理系统

## 项目介绍
一个基于 HTML+CSS+JavaScript 实现的前端图书管理系统，支持图书信息的增删改查、搜索筛选与本地数据持久化，适合前端初学者练手。

## 核心功能
- ✅ 展示图书列表（书名、作者、类型、状态）
- ✅ 添加/编辑/删除图书信息
- ✅ 按书名/作者模糊搜索图书
- ✅ 本地存储（localStorage），刷新页面数据不丢失

## 技术栈
- HTML5：页面结构搭建
- CSS3/Bootstrap：界面样式与响应式布局
- JavaScript (ES6)：交互逻辑与数据处理
- localStorage：浏览器本地数据持久化

## 运行方式
直接打开 `index.html` 文件，即可在浏览器中运行项目，无需后端与数据库。

## 项目亮点
- 代码结构清晰，核心逻辑（增删改查）完整
- 支持模糊搜索，提升用户体验
- 表单校验，避免空数据提交
- 界面简洁美观，操作流程友好
